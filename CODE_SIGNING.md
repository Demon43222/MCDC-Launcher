# Code Signing

Free code signing is provided by SignPath.io, certificate by SignPath Foundation.

Windows release artifacts of MCDC.pl Launcher may be signed using a certificate issued to SignPath Foundation.

## Signed artifacts

The following artifacts may be signed:

- Windows NSIS installer
- Windows launcher executable

## Verification

Users can verify the digital signature on Windows:

1. Right-click the downloaded `.exe` file.
2. Select `Properties`.
3. Open the `Digital Signatures` tab.
4. Check that the signer is `SignPath Foundation`.
